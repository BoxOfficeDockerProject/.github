
# 🍿FavieList
카카오 클라우드 스쿨에서 진행한 도커 기반의 3-Tire 프로젝트
* 목표 : Front, Back, DB의 멀티 컨테이너로 도커 환경에서 서비스를 운영
* 서비스 주제 : 관심영화 리뷰 사이트
* 서비스 기능 : 영화 검색, 관심 등록, 리뷰 CRUD
